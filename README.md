# Mobile Device Authentication Project

## Overview:
The Mobile Device Authentication project focuses on implementing a secure authentication mechanism for Android devices using a custom algorithm and secure store technique. This project aims to enhance the security of mobile devices by providing a robust authentication method that ensures only authorized users can access the device.


## Technologies Used:
- **Programming Language:** Java
- **Android SDK:** Android Development Kit
- **Security Libraries:** Android Keystore, AES Encryption
- **Authentication Mechanisms:** Custom Authentication Algorithm, Multi-factor Authentication
